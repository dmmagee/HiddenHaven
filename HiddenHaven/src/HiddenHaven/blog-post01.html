Welcome to the new web site for Hidden Haven Horse Care & Boarding Facility!

Thank you for stopping by and checking out exactly what Hidden Haven has to offer! This web site has been a long time coming, and this site launch certainly is exciting!

I designed this web site to help myself get practice as I become a new software developer, but more importantly to help my folks spread the success of their boarding business. This is the first web site I have ever designed and built, so if there are any bugs, or it looks like a 5th grader designed this back in 1998, please forgive me :)

My hope and goal is that this web site helps my folks business flourish even more than it already has, and to continue to develop skills and techniques while adding to the site.

Please stop by this community forum often to catch all the updates from us, and thank you so much for checkin us out!
